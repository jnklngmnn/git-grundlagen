# git-grundlagen
## Unterschied zwischen lokalem und remote Repository

- **Lokales Repository:** befindet sich auf meinem eigenen Computer. Hier arbeite ich direkt an den Dateien und Commits.
- **Remote Repository:** liegt auf einem Server (z. B. GitHub). Dorthin pushe ich meine Änderungen, um sie zu sichern oder mit anderen zu teilen.
